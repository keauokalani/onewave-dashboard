How to create a GitHub token (one-time setup)
1

Go to GitHub Token Settings

Open Fine-grained tokens in GitHub Settings

2

Generate new token

Click "Generate new token"

3

Configure the token

✓ Name: "OneWave Studio" (or anything)
✓ Expiration: Choose the maximum (1 year) or "Custom" with a far-future date. You'll get an email reminder before it expires.
✓ Repository access: Select "Only select repositories" and pick keauokalani/onewave-dashboard
✓ Permissions: Under "Repository permissions", set Contents to Read and write. That's it — just one permission.
4

Copy & paste

Copy the token (starts with github_pat_) and paste it above. If "Keep me logged in" is checked, you won't need to do this again.

Your token is stored only in your browser's localStorage and is sent only to GitHub's API. It is never sent to any other server. Use "Logout" below to clear it at any time.
